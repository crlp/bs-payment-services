# bs-payment-services

## Create docker image "docker build"
`docker build -t bs-payment-services:0.0.1 .`

`docker image ls`

## Crear contenedor "docker run"

`docker run -d --name bs-payment-services-dev -p 9092:8082 bs-payment-services:0.0.1`
